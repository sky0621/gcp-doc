# Google App Engine

  以下はマネージド
  - トラフィック処理目的でのアプリのスケールアップまたはスケールダウン
  - インスタンスの負荷分散
  - ヘルスチェック
  - インスタンスの復旧
  - 基盤 OS へのアップデートの適用
  
  開発言語
  - Node.js
  - Java
  - Ruby
  - C#
  - Go
  - Python
  - PHP
  
  特徴
  - バージョンの異なるアプリも簡単にホスティング
  - 受信したリクエストをさまざまなバージョンのアプリに転送して A/B テストを行い、機能を段階的に公開
  - カスタムドメインで SSL/TLS マネージド証明書* をデフォルトで利用可能（追加費用なし）
